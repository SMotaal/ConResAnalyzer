ConResAnalyzer

This software is designed to simplify the job of conducting analysis of the Contrast-Resolution (ConRes) quality attributes of a monochromatic printing system.

ConRes is analyzed using special test targets designed by Franz Sigg (@FranzSigg) using hundreds of square-wave concentric circle patches at various frequencies, tone and contrast levels. The analysis involves systematic assessment of the visibility thresholds for the resolution, tone and contrast extremes to determine the limitations of the printing system.

Numeric and visual representations are produced to document the findings in the final analysis report. Systems are characterized for quality control purposes and ranked for comparisons against ideal baselines related relative to the response the human visual system.

Core Functions

The goal is to create an automated software that requires minimal operator intervention. This eliminates the redundant tasks which are handled systematically. To accomplish this, modules are developed for digital image segmentation, Fourier image analysis, and search algorithms.

Java is used to develop the final solution. MatLab is also used extensively in the research and development phase in order to refine the algorithms used.
